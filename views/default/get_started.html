Get Started - Instructions

This page is a quick overview of how Medboard works.

There are 2 parts to Medboard:

{{=IMG(_src=URL('static','images/menu_bar.png'))}}
  • Browse – this is where you can see what teaching sessions are available and sign up
  • My Sessions – this is where you can see what sessions you have registered to attend

Using the site

To access the site go to: http://medboard.co.uk

You can access it on any device with a web browser and it should format itself to fit on any screen (eg your smartphones)

To make best use of the site you need to register. You can do this {{=A('here',_href=URL('user',args=['register']))}}

When registering please complete all the boxes except the Access Key.

The Default Hospital box should be completed to match your placement block.

For example, if you are starting Oncology at the Royal Glamorgan hospital select “RGL – Oncology”.

Abbreviations are occasionally used for hospital names.

  • RGL – Royal Glamorgan
  • PCH – Prince Charles

Browse

Once you are registered go to {{=A('Browse',_href=URL('browse'))}} to see what sessions are available.

You will be shown the sessions related to your block by default if you are logged in.

The dropdown box will let you change the view to see other blocks if you want.

{{=IMG(_src=URL('static','images/hospital_dropdown.png'))}}

In the table you will see a different session list on each row.

{{=IMG(_src=URL('static','images/browse.png'))}}

By default, 10 sessions will be shown on each page, but you can change that using the dropdown. To search for something specific, use the search box.

Each session has the following attributes:

  • Session Type – This is either Clinic, MDT, Theatre, Teaching Session or Allied Health Professionals
  • Title – This is the name of the session
  • Start Date and Time
  • Session Lead Name
  • Status – Shows you how many space are available in the session
  • View Button – this takes you to a page showing a few more details for that session
  • Sign Up Button – clicking this will add your name to the session

My Sessions

This can be found {{=A('here',_href=URL('my_sessions'))}}

  • Upcoming Sessions – these are sessions in the future that you have signed up to

  • ....The feedback button here will let you leave comments on the session
